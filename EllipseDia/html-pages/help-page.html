Возможности редактора

Редактор позволяет строить диаграммы:

  • Диаграмма вариантов использования (Диаграмма прецедентов, ДВИ, Use Case Diagram)
  • Диаграмма состояний (Statechart Diagram)
  • Диаграмма деятельности/активностей (Activity Diagram)
  • Диаграмма классов (Class Diagram)
  • Диаграмма объектов (Object Diagram)

Секции редактора

Редактор состоит из следующих секций:

  1. Строка заголовока — содержит название диаграммы, а также ссылки на страницу аккаунта и справку.
  2. Главные кнопки — содержит кнопки, отвечающие за работу с буфером обмена, экспортом, сохранением и т.д.
  3. Секция выделения — содержит кнопки, отвечающие за выделения элементов на диаграмме.
  4. Панель элементов — содержит элементы диаграммы (соединения и блоки).
  5. Окно диаграммы — содержит "полотно", на котором размещаются элементы диаграммы.
  6. Контекстное меню — содержит кнопки, отвечающие за более детальное изменение блока диаграммы.
  7. Панель свойств — содержит форму, отображающую информацию о свойствах блока диаграммы.
  8. Секция масштабирования — содержит кнопки, отвечающие за изменение масштаба диаграммы.
  9. Флаг подбора цвета — открывает небольшое окно, в котором можно удобно подобрать цвет для блока диаграммы.
  10. Флаг кода диаграммы — показывает код диаграммы, который можно изменить.

Список горячих клавиш

  • Ctrl + G — Группировать
  • Ctrl + U — Разгруппировать
  • Ctrl + C — Копировать
  • Ctrl + X — Вырезать
  • Ctrl + V — Вставить
  • Ctrl + A — Выделить всё
  • Ctrl + Z — Отменить действие
  • Ctrl + Y — Вернуть действие

  • Стрелка вниз — Выйти из родительского блока
  • Стрелка вверх — Войти в родительский блок

  • F2 — Редатировать
  • F12 — Открыть свойства
  • + — Ближе
  • - — Дальше

Создавайте UML-диаграммы онлайн

EllipseDia позволяет быстро создавать красивые UML-диаграммы. Создавайте диаграммы легко с помощью редактора: перетаскивайте элементы, проектируйте с помощью разнообразного набора символов UML, храните свои разработки в облачном рабочем пространстве и делитесь ими. Вы также можете экспортировать диаграмму в XML, SVG и PNG.
